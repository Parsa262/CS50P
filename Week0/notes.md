# CS50P Notes

## Day 1

* Python runs through the terminal.
* `print()` → prints text or values to the console.
* Variables → store values (e.g. `name = "Parsa"`).
* `input()` → gets input from the user (returns a string by default).
* Strings (`str`) → text.
* f-strings → insert variables or expressions into a string using `{}`.
* Named parameters → arguments passed by name, e.g. `input(prompt="Name: ")`

## Day 2

* `int` → whole numbers.
* `float` → decimal numbers.
* String methods → functions that work on strings, e.g. `.strip()`, `.lower()`, `.upper()`, `.title()`.
* Operators:

  * `+` addition
  * `-` subtraction
  * `*` multiplication
  * `/` division (returns a float)
  * `%` remainder (modulo)
* Formatting numbers:

  * `f"{x:,}"` adds thousands separators.
  * `f"{x:.2f}"` shows 2 decimal places.
* Division and rounding:

  * `round(x)` rounds to the nearest integer.
  * `round(x, 2)` rounds to 2 decimal places.
* `def` → defines (creates) a function.
* Scope → variables created inside a function usually exist only inside that function.
* `return` → sends a value back from a function so it can be used elsewhere.
